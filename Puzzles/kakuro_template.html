What the techniques mean
Last cellT1
Only one cell of a run is empty — it must be whatever remains of the sum.
Forced singleT2
Only one candidate digit is left for a cell; place it.
Sum boundsT3
Quick arithmetic: a digit is impossible when the rest of the run couldn't be small or large enough to finish the sum with distinct digits.
Unique combinationT4
Some sums decompose only one way (the "magic blocks" — e.g. two cells summing 3 must be {1,2}; four cells summing 30 must be {6,7,8,9}). Every other digit is struck from the run.
Combination filterT5
List every digit-set that could still complete the run; digits in none of them are struck.
Required digitT6
A digit appears in every remaining combination for a run, and only one cell can host it — place it there.
Naked pairT7 & Naked tripleT8
Two (or three) cells of a run share the same two (three) candidates; those digits are used up and are struck from the run's other cells.
Run analysisT9
Work through every arrangement of a run that respects the current pencil marks; digits appearing in no arrangement are struck. The heavy machinery of hard kakuro.
