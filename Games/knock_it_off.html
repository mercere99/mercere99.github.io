You can push an item only when you stand at its height, and only onto a tile of equal or lower height. You, however, can jump up to anything. You're a cat.
How to add your own levels, music, and sounds

Levels live in the LEVELS array near the top of the script in this file. Each level is two aligned ASCII grids: heights (digits 0–9 = tile height, # = solid wall, space = no tile) and objects (@ cat, $ item, x target, * item already on a target, + cat on a target, . empty). A # in either grid makes a wall.

Music: set MUSIC_SRC to a file path or URL (e.g. "music/lofi_cat.mp3") and it will loop; toggle with the 🎵 button.

Sound effects: every effect is synthesized, so the game needs no assets — but any entry in CUSTOM_SOUNDS that you point at an audio file replaces the corresponding synth.

Achievements are defined in the ACHIEVEMENTS array; add an entry and call award("yourId") from wherever its condition is detected.

Full details are in the comment block at the top of the script.
